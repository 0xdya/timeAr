# ArTime

مكتبة بسيطة لتحويل التواريخ إلى صيغة "منذ..." بالعربية، مع دعم التصريف الصحيح للأعداد.

---

## cdn
ضع في نهاية head او نهاية body هذا
```html
<script src="https://cdn.jsdelivr.net/gh/0xdya/ArTime@cdn/dist/artime.js"></script>
```

ثم الاستعمال يكون
```html
<!-- طريقة 1: data-date (يبقى العنصر فارغاً أو بأي نص) -->
<span class="time_ago" data-date="2025-01-15T10:00:00"></span>

<!-- طريقة 2: التاريخ مباشرة في النص -->
<span class="time_ago">2025-01-15T10:00:00</span
```

---
## npm
```
npm install time-ar
```
---
لماذا ArTime؟
* دعم العربية الفصحى: تهتم المكتبة بالتفاصيل اللغوية (ثانية، ثانيتان، ثوانٍ).
* خفيفة جداً: لا تعتمد على مكتبات ضخمة مثل Moment.js.
* مرونة عالية: تعمل مع وسوم HTML مباشرة دون الحاجة لكتابة كود JS إضافي في أغلب الحالات.
